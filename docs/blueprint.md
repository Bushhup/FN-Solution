# **App Name**: FN Tax Solution

## Core Features:

- Service Catalog: Display a structured list of tax, compliance, business registration, legal, and accounting services with detailed information and request forms.
- Lead Capture Forms: Collect user information (name, email, phone, service selection) through various forms (service request, consultation request, contact form, newsletter signup). Store data in a database and trigger notification emails.
- User Authentication: Enable user registration and login with email/password, including password reset functionality. Implement secure password hashing and role-based access (User/Admin).
- User Dashboard: Provide a user dashboard for logged-in users to view their submitted service requests, update their profile, and track the status of their requests.
- Admin Panel: Create a secure admin panel to manage services, leads, customers, admin users, and testimonials. Provide functionalities to view leads, update lead status, add/edit/delete services, manage testimonials, and view basic analytics.

## Style Guidelines:

- Primary color: Deep Sky Blue (#3399FF) for a professional and trustworthy feel.
- Accent color: Vivid Cyan (#00FFFF), offering contrast for highlights.
- Dark background: #0F172A (primary background).
- Light background sections: #F0F2F5.
- Headline font: 'Space Grotesk', bold, slightly letter-spaced.
- Body font: 'Inter', clean and readable.
- Use professional icons (Lucide/Heroicons) for service categories and navigation.
- Maintain a clean grid layout with clear spacing and breathing room.
- Incorporate subtle animations such as fade-in on scroll, button hover lift effect, and glow pulse on CTA. Use smooth page transitions.
